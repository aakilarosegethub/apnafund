# KickStarted - Header and Banner Design

A modern, responsive header and banner section designed to mimic Kickstarter's aesthetic using Bootstrap 5 and the Inter font family.

## Features

- **Modern Header Design**: Clean navigation with dropdown menus, search functionality, and user actions
- **Hero Banner**: Eye-catching hero section with compelling copy and call-to-action buttons
- **Featured Projects**: Showcase section displaying project cards with funding progress
- **Responsive Design**: Fully responsive across all device sizes
- **Inter Font Family**: Uses Google Fonts' Inter for a modern, clean typography
- **Bootstrap 5**: Latest version of Bootstrap for robust component styling
- **Smooth Animations**: Hover effects and transitions for enhanced user experience

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styling with modern CSS features
- **Bootstrap 5.3.2**: Latest Bootstrap framework
- **Inter Font**: Google Fonts' Inter font family
- **Font Awesome 6.4.0**: Icon library for search and UI elements

## Key Design Elements

### Header Section
- Fixed navigation bar with subtle shadow
- Logo placement on the left
- Centered navigation menu with dropdown
- Search bar with rounded design
- Login/Signup buttons on the right

### Hero Banner
- Large, bold typography using Inter font
- Compelling headline and description
- Statistics display showing platform metrics
- Dual call-to-action buttons
- Hero image with rounded corners and shadow

### Featured Projects
- Project cards with hover effects
- Progress bars showing funding status
- Clean typography and spacing
- Responsive grid layout

## Color Scheme

- **Primary Blue**: #0066cc (Kickstarter's signature blue)
- **Text Colors**: #000 (black), #333 (dark gray), #666 (medium gray)
- **Background**: #fff (white), #f8f9fa (light gray)
- **Borders**: #e5e5e5 (light gray)

## Typography

- **Font Family**: Inter (Google Fonts)
- **Font Weights**: 300, 400, 500, 600, 700, 800, 900
- **Headings**: Bold weights (700-800) with negative letter-spacing
- **Body Text**: Regular weight (400) with optimal line-height

## Responsive Breakpoints

- **Desktop**: 992px and above
- **Tablet**: 768px - 991px
- **Mobile**: Below 768px
- **Small Mobile**: Below 576px

## Usage

1. Open `index.html` in a web browser
2. The page will load with all styles and functionality
3. All external resources are loaded via CDN (no local files needed)

## Customization

To customize the design:

1. **Colors**: Modify the CSS custom properties in `styles.css`
2. **Typography**: Adjust font sizes and weights in the CSS
3. **Layout**: Modify Bootstrap classes in the HTML
4. **Content**: Update text content in the HTML file
5. **Images**: Replace placeholder images with your own assets
